# 카카오 비즈니스 채팅창 크기 조절 확장 프로그램

카카오톡 비즈니스 채팅창의 크기를 자유롭게 조절할 수 있는 Chrome 확장 프로그램입니다.

## 기능

- 📏 채팅 입력창 크기 자유롭게 조절
- 🖱️ 드래그로 간편하게 크기 변경
- 💾 설정한 크기 자동 저장
- 🔄 더블클릭으로 기본 크기 복원

## 설치 방법

1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 "개발자 모드" 활성화
3. "압축 해제된 확장 프로그램을 로드합니다" 클릭
4. 이 폴더 선택

## 사용 방법

1. 카카오 비즈니스 페이지에 접속
2. 채팅 입력창 상단에 나타나는 조절 바를 드래그하여 크기 조절
3. 더블클릭하면 기본 크기로 복원됩니다

## 파일 구조

- `manifest.json` - 확장 프로그램 설정 파일
- `content.js` - 크기 조절 기능 스크립트
- `content.css` - 스타일 파일
- `icon16.png`, `icon48.png`, `icon128.png` - 확장 프로그램 아이콘 (추가 필요)

## 아이콘 생성 방법

아이콘 파일을 직접 만들거나 다음 사이트에서 생성할 수 있습니다:
- https://www.favicon-generator.org/
- https://favicon.io/

또는 간단히 텍스트 편집기로 SVG를 만들어 PNG로 변환할 수 있습니다.

## 주의사항

- 이 확장 프로그램은 카카오 비즈니스 페이지에서만 작동합니다.
- 최소 높이: 60px, 최대 높이: 800px로 제한되어 있습니다.
