# 카카오 비즈니스 유틸리티

카카오톡 비즈니스 관리 페이지에 유용한 기능을 추가하는 Chrome/Firefox 확장 프로그램입니다.

## 주요 기능

### 1. 채팅창 크기 조절
- 채팅 입력창(`.write_chat3`)의 높이를 자유롭게 조절할 수 있습니다
- 상단의 드래그 핸들을 위아래로 드래그하여 크기를 조절합니다
- 조절한 크기는 로컬스토리지에 자동으로 저장되어 다음 방문 시에도 유지됩니다
- 더블클릭으로 기본 크기로 되돌릴 수 있습니다
- 최소 60px, 최대 800px 범위 내에서 조절 가능합니다

### 2. 채팅 로그 미리보기
- 채팅 리스트의 항목에 마우스를 1초 이상 올리면 해당 채팅의 이전 대화 내용을 미리볼 수 있습니다
- 팝업은 화면 우측에 고정되며, 스크롤 시 항목을 따라 이동합니다
- 팝업 외부 클릭 또는 닫기 버튼(✕)으로 닫을 수 있습니다
- 채팅 항목이 DOM에서 제거되면 팝업도 자동으로 닫힙니다
- 최신 메시지가 하단에 표시되며 자동으로 스크롤됩니다

### 3. 설정 관리
- 확장 프로그램 아이콘을 클릭하여 설정 팝업을 열 수 있습니다
- 각 기능을 개별적으로 켜고 끌 수 있습니다
- 설정은 Chrome Sync Storage에 저장되어 여러 기기에서 동기화됩니다

## 디자인

모노크롬 디자인으로 통일되어 있으며, 카카오 비즈니스 인터페이스와 자연스럽게 어우러집니다:
- 회색 톤의 미니멀한 디자인
- 호버 및 드래그 시 그라데이션 효과로 시각적 피드백 제공
- 부드러운 애니메이션 전환

## 설치 방법

### Chrome
1. 이 저장소를 클론하거나 다운로드합니다
2. Chrome에서 `chrome://extensions/` 페이지를 엽니다
3. 우측 상단의 "개발자 모드"를 활성화합니다
4. "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭합니다
5. 이 프로젝트 폴더를 선택합니다

### Firefox
1. 이 저장소를 클론하거나 다운로드합니다
2. Firefox에서 `about:debugging#/runtime/this-firefox` 페이지를 엽니다
3. "임시 부가 기능 로드" 버튼을 클릭합니다
4. 프로젝트 폴더에서 `manifest.json` 파일을 선택합니다

## 사용 방법

1. 확장 프로그램을 설치합니다
2. [카카오 비즈니스](https://business.kakao.com) 페이지에 접속합니다
3. 채팅 입력창 상단의 드래그 핸들을 이용해 크기를 조절할 수 있습니다
4. 채팅 리스트의 항목에 마우스를 올려 채팅 로그를 미리볼 수 있습니다
5. 확장 프로그램 아이콘을 클릭하여 기능을 켜고 끌 수 있습니다

## 파일 구조

```
kakao-business-size-adjustment/
├── manifest.json          # 확장 프로그램 설정 파일 (Manifest V3)
├── content.js             # 메인 기능 구현 (리사이즈, 채팅 로그)
├── content.css            # 스타일 정의
├── popup.html             # 설정 팝업 UI
├── popup.js               # 설정 팝업 로직
├── create-icons.html      # 아이콘 생성 도구
├── icon16.png             # 16x16 아이콘
├── icon48.png             # 48x48 아이콘
├── icon128.png            # 128x128 아이콘
└── README.md              # 프로젝트 문서
```

## 기술 스택

- **Manifest V3**: 최신 Chrome 확장 프로그램 표준
- **Chrome Storage API**: 설정 동기화
- **Chrome Messaging API**: 팝업-콘텐츠 스크립트 간 통신
- **MutationObserver**: DOM 변경 감지 및 동적 요소 처리
- **LocalStorage**: 리사이즈 크기 저장
- **Fetch API**: 채팅 로그 데이터 가져오기

## 브라우저 호환성

## 문제 해결

### 리사이즈 핸들이 보이지 않아요
- 설정 팝업에서 "채팅창 크기 조절" 기능이 활성화되어 있는지 확인하세요
- 페이지를 새로고침해보세요

### 채팅 로그가 나타나지 않아요
- 설정 팝업에서 "채팅 로그 미리보기" 기능이 활성화되어 있는지 확인하세요
- 최소 1초 이상 항목에 마우스를 올려두세요
- 네트워크 연결 상태를 확인하세요

### 설정이 저장되지 않아요
- Chrome/Firefox의 동기화 기능이 활성화되어 있는지 확인하세요
- 확장 프로그램 권한에서 `storage` 권한이 허용되어 있는지 확인하세요
